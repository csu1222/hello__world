0. 수업소개 - 비쥬얼스튜디오코드를 사용하여 git 을 조작 하는 수업 입니다. 또한 버전 관리 프로그램을 잘 사용하면 디버깅에 유리 합니다. 좋은 버전이란 여러 작업을 같이 묶지 않고 한 버전에 한 작업을 관리 하는 것 이고 언제든 버전을 찾을 수 있도록 그 버전을 잘 설명하는 표현이 있어야 합니다.
1. 저장소 만들기 - 자신이 작업을 시작 할 프로젝트 디렉토리를 만들고 VSC로 해달 폴더를 엽니다. 좌측 위에서 3번째 아이콘으로 리포지토리 초기화를 하면 프로젝트 디렉토리에 .git 폴더가 생성됩니다. (.git 폴더에 git 으로 버전관리 데이터가 들어있습니다.)
2. 버전 만들기 - 프로젝트 디렉토리에 작업을 진행한 후 VSC의 소스 제어 아이콘으로 가면 아직 detached 되지 않은 파일들이 있습니다. 파일을 stageing 하고 commit 하면 staging 된 파일들을 .git 폴더에 하나의 버전으로 commit 하게 됩니다.
3. 버전 보기 (git graph) - VSC의 확장 프로그램으로 git graph를 설치하게 되면 git log 를 GUI적으로 직관적으로 확인할 수 있게 됩니다.
4. 버전 사이의 이동 - command line 에서 git checkout (commit id 또는 branch name)으로 해당 버전으로 이동 합니다. 또는 VSC에 git graph가 설치 되어있으면 GUI 상 해당 버전을 우클릭 하여 checkout 으로 이동 할 수 있습니다.
5. 또 다른 효용들 - 파일들의 변경점확인, 변경점 되돌리기 를 할 수 있습니다.
